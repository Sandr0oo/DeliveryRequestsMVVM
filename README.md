# Описание DeliveryRequestsMVVM
Тестовая программа не несет никакого практического применения, но отражает мои навыки в области программирования. 
## Возможности программы
- Добавить заявку
- Удалить заявку
- Поиск заявки по всем полям
- Изменить статус заявку.

# Технологии в проекте
- .NET Core 3.1
- Примненен паттерн MVVM
- Entity Framework 
- База данных SQLite
